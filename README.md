## 🖥️ Projeto
O projeto Web Responsivo de uma capsula do tempo para exibir memôrias em uma linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

-HTML
-CSS
-Git e Github
